# Node CLI Typescript

a simple boilerplate node cli written in typescript.

## Setup

install deps:

```bash
$ npm i
```

create entrypoint:

```bash
# grant perms
$ chmod u+x ./src/index.ts
```

test:

```bash
# test
$ ./src/index.ts
```

## Publish

publish to npm:

```
$ npm publish
```

install:

```bash
$ npm install -g <node-typescript-cli>
```

test:

```bash
$ <node-typescript-cli>
```
